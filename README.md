# BAKEET

A lightweight data export and retrieval demo for local AI analysis.

## 🚀 Project Overview

This repository contains a simple project that stores and exposes local AI retrieval results in a compact format.

## 📁 Repository Structure

- `bakeet_results.html` — an HTML report or summary page for the project.
- `LocalAIRetrival/LocalAIService_BAKEET.csv` — raw CSV output from local AI retrieval.
- `LocalAIRetrival/LocalAIService_BAKEET.ods` — spreadsheet version of the same local AI result data.

## ✨ Features

- Local AI retrieval results are preserved in both CSV and ODS formats.
- Simple HTML summary page for quick review.
- Minimal repository structure for easy sharing and version control.

## 🛠️ How to Use

1. Open `bakeet_results.html` in a browser to review the result summary.
2. Use the CSV or ODS files in `LocalAIRetrival/` for data analysis in Excel, LibreOffice, or other spreadsheet tools.
